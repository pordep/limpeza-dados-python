# Limpeza de Dados para Crédito

Script para praticar o que um Engenheiro de Dados faz no dia a dia. 

### Atividades:
* **Limpeza de dinheiro:**  replace e Regex para tirar o cifrão e trocar a vírgula por ponto.
* **Tratamento de erros:** `errors='coerce'` no Pandas.
* **Mediana vs Média:** **Mediana** para preencher os valores vazios para ignorar esses valores absurdos.
* **Lambda:** Função de uma linha só para marcar o que é valor suspeito (acima de 50k).

### Objetivos:
Garantir que segurança do código e que os cálculos fizessem sentido para o negócio, sem que um único valor alto estragasse toda a média dos clientes.
